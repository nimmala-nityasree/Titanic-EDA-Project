# Exploratory Data Analysis (EDA) on Titanic Dataset

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, and factors influencing passenger survival.

## Dataset Overview

The Titanic dataset contains information about passengers, including age, gender, passenger class, fare, family members aboard, and survival status.

## Statistical Summary

Descriptive statistics were generated using the dataset to understand the distribution of numerical features such as Age, Fare, SibSp, and Parch.

## Visualizations and Analysis

### 1. Survival Distribution

A bar chart was created to analyze the distribution of survivors and non-survivors. The analysis showed that the number of passengers who did not survive was higher than the number of survivors.

### 2. Survival by Gender

A comparison between gender and survival status revealed that female passengers had a significantly higher survival rate than male passengers.

### 3. Survival by Passenger Class

Passenger class was found to be an important factor. First-class passengers had a higher probability of survival compared to passengers in second and third class.

### 4. Correlation Analysis

A correlation heatmap was generated to identify relationships between numerical variables. The analysis indicated that passenger class, fare, and gender were among the key factors influencing survival.

## Key Findings

* More passengers died than survived.
* Female passengers had a higher survival rate.
* First-class passengers had better survival chances.
* Higher fare values were generally associated with increased survival probability.
* Passenger class and gender were significant influencing factors.

## Conclusion

Exploratory Data Analysis helped uncover meaningful patterns within the Titanic dataset. Gender, passenger class, and fare were identified as major factors affecting passenger survival. These insights provide a better understanding of the dataset and demonstrate the importance of data exploration before building predictive models.
